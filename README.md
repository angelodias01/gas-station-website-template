EcoFuel Station Website Template
This repository contains a responsive, single-page landing page designed specifically for gas stations, petrol bunkers, and electric vehicle (EV) charging hubs.

The project is built using standard HTML5, CSS3, and JavaScript, making it lightweight and easy to customize without the need for external frameworks or build tools.

Project Overview
The template provides a professional online presence for a fuel station. It includes sections for displaying available fuel types, company information, customer reviews, and contact details with an integrated map.

Key Features
Responsive Design: The layout automatically adjusts to fit mobile phones, tablets, and desktop screens.

Service Grid: A visual display of available fuel types (Unleaded, Diesel, LPG) and EV charging status.

Custom Graphics: Includes a custom SVG illustration embedded directly in the code, ensuring fast loading times without external image dependencies.

Map Integration: Integrated OpenStreetMap embed to display the station location (no API key required).

Interactive Elements: Includes a mobile navigation menu and smooth scrolling for a better user experience.

Technologies Used
HTML5: For semantic page structure.

CSS3: Uses Flexbox and CSS Grid for layout, along with CSS Variables for easy color customization.

JavaScript: Vanilla JS used for the mobile menu toggle and smooth scrolling behavior.

How to Run
Download or clone this repository to your local machine.

Navigate to the project folder.

Double-click the index.html file to open it in your default web browser.

No server setup or installation process is required.

File Structure
index.html: Contains all the text, structure, and the custom SVG image code.

style.css: Contains all styling rules, colors, and responsive media queries.

script.js: Handles the interactivity for the mobile navigation bar and scroll effects.

Customization Guide
Changing Colors: Open style.css and look for the :root section at the top. Change the hex codes for --primary-color to match your brand.

Changing Text: Open index.html and edit the text content within the tags.

Replacing Images: Update the src attributes in the HTML to point to your own image files.
